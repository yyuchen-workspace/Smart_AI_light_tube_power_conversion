## 專案概述
智能燈管節電計算應用程式，提供三步驟的電費試算與節電回本分析。

**目前版本**：v2.1（整合版）

**主要功能**：
- ✅ Step 1：AI 智能燈管耗電計算（含傳統燈管欄位）
- ✅ Step 2：台電電費計算與分析
- ✅ Step 3：節電回本期間試算（租賃/買斷模式）
- ✅ 響應式設計（桌面雙欄 + 手機單欄佈局）
- ✅ 互動式圖表（圓餅圖、折線圖）
- ✅ 跨午夜時段計算修正

### 7. **lib/utils/electricity_calculator.dart** (電費計算邏輯)

**主要職責**：
- 實作台電電費計算公式

**主要函數**：

#### `double calculateBasicElectricity(double contractCapacity, bool isSummer)`
- **功能**：計算基本電價（約定）
- **公式**：
  ```dart
  夏季：contractCapacity * ElectricityPricing.summerBasicRate
  非夏季：contractCapacity * ElectricityPricing.nonSummerBasicRate
  ```
- **回傳**：基本電價（元）

#### `double calculateExcessDemand(double maxDemand, double contractCapacity, bool isSummer)`
- **功能**：計算最高需量超約費用
- **公式**：
  ```dart
  if (maxDemand > contractCapacity) {
    excess = maxDemand - contractCapacity
    夏季：excess * ElectricityPricing.summerExcessRate * 2
    非夏季：excess * ElectricityPricing.nonSummerExcessRate * 2
  }
  ```
- **回傳**：超約費用（元）

#### `double calculateFlowElectricity(double billingUnits, bool isSummer)`
- **功能**：計算流動電價
- **公式**（分級累進）：
  ```dart
  夏季級距：
    0-330度：2.10元/度
    331-700度：2.89元/度
    701-1500度：3.94元/度
    1501度以上：5.17元/度

  非夏季級距：
    0-330度：2.10元/度
    331-700度：2.62元/度
    701-1500度：3.61元/度
    1501度以上：4.01元/度
  ```
- **回傳**：流動電價（元）

### 9. **lib/utils/lighting_calculator.dart** (燈管耗電計算邏輯)

**主要職責**：
- 計算車道燈/車位燈的每日耗電量
- 處理日夜分段與全天候模式
- 計算感應亮度變化

**主要函數**：

#### `double calculateDrivewayWattage(LightingStrategy strategy)`
- **功能**：計算車道燈每日耗電（Wh）
- **邏輯**：
  1. 判斷全天候或日夜分段模式
  2. 對每個時段：
     ```dart
     baseWattage = duration * beforeSensingWattage
     sensingWattage = sensingCount * sensingHours * (afterWattage - beforeWattage)
     totalWattage = baseWattage + sensingWattage
     ```
  3. 車道燈感應次數：
     - 日間：1440次/天
     - 夜間：885次/天
- **回傳**：每日耗電（Wh）

#### `double calculateParkingWattage(LightingStrategy strategy)`
- **功能**：計算車位燈每日耗電（Wh）
- **邏輯**：同車道燈，但感應次數不同
  - 日間：80次/天
  - 夜間：30次/天
- **回傳**：每日耗電（Wh）

**輔助函數**：
- `_calculateTimeSlotWattage()`：計算單一時段耗電
- `_getSensingCount()`：取得感應次數

### ✅ **已解決：跨午夜時段計算錯誤（2025-01）**

**症狀**：
- 當時段設定為 23:00-01:00 時，計算結果錯誤
- duration 計算為負值（-22 小時）

**根本原因**：
- `TimeSlotConfig.duration` getter 未處理 `endHour < startHour` 情況

**解決方案**：
```dart
double get duration {
  if (isAllDay) return 24.0;
  double diff = endHour - startHour;
  return diff < 0 ? diff + 24 : diff;  // 修正跨午夜計算
}
```

## 🏗️ 技術架構設計

### **📊 雙模式電費計算系統**

本系統提供兩種電費計算模式，可透過 `main.dart` 中的開關切換：

```dart
/// 【計算模式切換】
/// true = 使用台電複雜計算（基本電價 + 流動電價 + 超約費用）
/// false = 使用簡化計算（度數 × 平均電價）
static const bool USE_COMPLEX_ELECTRICITY_CALCULATION = false;
```

#### **模式 A：簡化計算（預設，`false`）**

**計算公式**：
```dart
更換前每月電費 = 舊燈管耗電度數 × 平均電價
更換後每月電費 = AI燈管耗電度數 × 平均電價

平均電價：
- 夏季：4.08 元/度
- 非夏季：3.87 元/度
```

**優點**：
- ✅ 計算簡單直觀
- ✅ 不需要輸入契約容量等複雜資料
- ✅ 適合快速估算

**顯示位置**：
- Step 1 右側圖表顯示完整電費資訊
- Step 2 側邊欄顯示提示「目前使用簡化計算模式」

---

#### **模式 B：台電複雜計算（`true`）**

**計算公式**：
```dart
基本電費 = 契約容量 × 基本電價（夏季236.2元/瓩，非夏季173.2元/瓩）
超約費用 = max(0, 最高需量 - 契約容量) × 超約費率 × 2
流動電費 = 計費度數 × 流動電價（夏季4.08元/度，非夏季3.87元/度）
總電費 = 基本電費 + 超約費用 + 流動電費
```

**優點**：
- ✅ 精確符合台電實際計費方式
- ✅ 可計算超約罰款
- ✅ 適合有完整台電帳單資料的用戶

**顯示位置**：
- Step 2 需輸入契約容量、最高需量、計費度數
- Step 2 側邊欄顯示電費組成圓餅圖
- Step 3 使用 Step 2 計算的總電價進行回本試算

---

#### **如何切換模式**

1. 打開 `lib/main.dart`
2. 找到第 56 行的 `USE_COMPLEX_ELECTRICITY_CALCULATION` 常數
3. 修改值：
   - `false`：使用簡化計算
   - `true`：使用台電複雜計算
4. 重新執行應用程式

### **計算公式速查**

#### Step 1：燈管耗電計算
```dart
// 單一時段耗電（Wh）
基礎耗電 = 時段長度(h) × 感應前瓦數(W)
感應額外耗電 = 感應次數 × 感應時間(s) / 3600 × (感應後瓦數 - 感應前瓦數)
時段總耗電 = 基礎耗電 + 感應額外耗電

// 每月耗電（度）
每月耗電 = 每日耗電(Wh) × 30 / 1000
```

#### 電費計算（雙模式）

**簡化模式**（預設，`USE_COMPLEX_ELECTRICITY_CALCULATION = false`）：
```dart
更換前每月電費 = 舊燈管耗電度數 × 平均電價
更換後每月電費 = AI燈管耗電度數 × 平均電價
每月節省電費 = 更換前電費 - 更換後電費

平均電價：夏季 4.08 元/度，非夏季 3.87 元/度
```

**複雜模式**（`USE_COMPLEX_ELECTRICITY_CALCULATION = true`）：
```dart
基本電費 = 契約容量 × 基本電價（夏季236.2元/瓩，非夏季173.2元/瓩）
超約費用 = max(0, 最高需量 - 契約容量) × 超約費率 × 2
流動電費 = 計費度數 × 流動電價（夏季4.08元/度，非夏季3.87元/度）
總電費 = 基本電費 + 超約費用 + 流動電費
```

#### Step 3：回本計算（版本 10.0 組合計算）
```dart
// 租賃模式
每月燈管租金 = 燈管數量 × 燈管租賃單價
每月網關租金 = 網關數量 × 網關租賃單價
每月總租金 = 每月燈管租金 + 每月網關租金
每月淨利 = 每月可節省電費 - 每月總租金

// 買斷模式
燈管買斷總價 = 燈管數量 × 燈管買斷單價
網關買斷總價 = 網關數量 × 網關買斷單價
買斷總價 = 燈管買斷總價 + 網關買斷總價
回本期間(月) = 買斷總價 / 每月可節省電費

// 每月可節省電費（來自 Step 2）
每月可節省電費 = 更換前每月電費 - 更換後每月電費
```
